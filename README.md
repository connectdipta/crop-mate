## üåü Project Overview

**CropMate** is an innovative smart farming platform designed to revolutionize agriculture by connecting farmers with essential resources and intelligent tools. This frontend application provides a user-friendly interface for a suite of services aimed at optimizing farm management, boosting crop yields, and simplifying access to critical agricultural needs.

### Key Features:

* **Labor & Machinery Hiring:** Farmers can easily browse, book, and hire skilled laborers and essential machinery (tractors, harvesters, etc.) available in their area.
* **Agricultural Marketplace:** A dedicated section to purchase high-quality seeds, fertilizers, pesticides, and other agricultural medicines from verified suppliers.
* **AI Crop Disease Detection:** Upload photos of affected crops for instant AI-driven analysis to identify diseases and receive recommended treatments or medicines.
* **Smart Crop Suggestions:** Leverage AI to get personalized recommendations on the most suitable crops to grow based on soil type, seasonal data, and regional climate conditions for maximum profitability.
* **Service Coverage Map:** An interactive map powered by Leaflet.js to visualize service centers and coverage areas across different districts.
* **Comprehensive FAQ Section:** Answers to frequently asked questions about CropMate's services and functionalities.
* **User Testimonials & Client Logos:** Showcase social proof and partnerships.

## üì¶ Installation & Setup

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js (LTS version recommended)
* npm or Yarn (or pnpm)

### Frontend Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/](https://github.com/)[your-github-username]/cropmate-frontend.git
    cd cropmate-frontend
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or yarn install
    # or pnpm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    # or yarn dev
    # or pnpm dev
    ```
    This will start the Vite development server, usually at `http://localhost:5173`.

### Backend Requirements

This frontend application expects a backend API to serve data for `serviceCenters.json` (or similar endpoints).
For development, you might be serving `serviceCenters.json` as a static file.
For a complete application, ensure your backend is running and accessible.
